# Planning Protocols

## Family & Growth Rules
- **Family Sizing:** Default to 5 large sweet potatoes (1 per person + 1 extra for growth).
- **Protein Boost:** Proactively increase protein in vegetarian or lighter meals (e.g., adding parmesan, increasing ricotta, or mashing chickpeas into bases).
- **Kid Preferences:** Always substitute Tahini with Lemon-Garlic Yogurt sauce.
- **Preparation:** For mashes, use unpeeled yellow potatoes for texture and nutrients.

## Variety Rules
- **Meal Type Variety:** Each week must feature a different meal format each night (e.g., only one flatbread, one stir-fry, or one heavy pasta).
- **Rotation:** Avoid repeating the same main protein/format combination two nights in a row.
